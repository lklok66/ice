System Requirements

IceGrid Admin is a JavaTM application supported a wide range of platforms including Windows XP, MacOS X and Linux.

In order to run IceGrid Admin you need:

•the IceGridGUI.jar file, usually installed in the bin or lib directory of your Ice installation. If you are using a Linux RPM installation, IceGridGUI.jar is installed in /usr/share/java/Ice-3.4.1.
•a Java Runtime Environment (JRE) v1.5.0 or later. For Windows, Linux and Solaris, we recommend using a Sun JRE or JDK, available here.

 

If you want to read IceGrid XML files from IceGrid Admin, you also need to have the icegridadmin command-line utility (version 3.4.1) in your PATH.
