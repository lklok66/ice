Service

A service represents an IceBox service loaded (or potentially loaded) within an IceBox server.

States

A service can be started or stopped within an IceBox server. However IceGrid is currently unaware of such state changes and IceGrid Admin always shows services with the same icon ().

Properties

The Service Properties panel shows:

Description

A free-text description of this service.

Properties

A table showing all the Ice properties of this service. These properties may come from template definitions, property sets, service-instance properties etc. They are all combined in this table.

Entry Point

The entry point for this service. This corresponds to the value of the IceBox.Service.service-name property.
