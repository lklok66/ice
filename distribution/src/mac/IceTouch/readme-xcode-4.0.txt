This installer installs the following components:

 - Command line SDK for Mac OS X in: /opt/IceTouch-1.1
 - Xcode 4.0 Cocoa SDK in: <Xcode>/SDKs/IceTouch-1.1/macosx.sdk
 - Xcode 4.0 iPhone OS 4.3 SDK in: <Xcode>/SDKs/IceTouch-1.1/iphoneos.sdk
 - Xcode 4.0 iPhone Simulator 4.3 SDK in: <Xcode>/SDKs/IceTouch-1.1/iphonesimulator.sdk
 - Xcode 4.0 Plug-in for use with Ice Touch in: /Library/Application Support/Developer/4.0/Xcode/Plug-ins/
 - Documentation in: <Xcode>/Documentation/IceTouch-1.1
 - Examples in: <Xcode>/Examples/IceTouch-1.1

<Xcode> refers to the folder where Xcode 4.0 is installed, /Developer by default. If you installed Xcode 4.0 in a different location, please adjust the target folder for the "iPhone / Cocoa SDKs" in the next dialog.

To uninstall Ice Touch 1.1.1, please use the uninstall script that accompanies the distribution.
