This installer installs the following components:

 - Xcode 4.0 Command line SDK in: /opt/IceTouch-1.1
 - Xcode 4.0 Cocoa SDK in: /Developer/SDKs/IceTouch-1.1/macosx.sdk
 - Xcode 4.0 iPhone iOS 4.3 SDK in: /Developer/SDKs/IceTouch-1.1/iphoneos.sdk
 - Xcode 4.0 iPhone Simulator iOS 4.3 SDK in: /Developer/SDKs/IceTouch-1.1/iphonesimulator.sdk
 - Xcode 4.0 Plug-in for use with IceTouch in: /Library/Application Support/Developer/4.0/Xcode/Plug-ins/
 - Documentation in: /Developer/Documentation/IceTouch-1.1
 - Examples in: /Developer/Examples/IceTouch-1.1

To uninstall IceTouch-1.1.1 please use the uninstall script that
accompanies the distribution.

For uninstall details execute:

> uninstall.sh -h
