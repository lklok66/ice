This installer installs the following components:

 - Xcode 4.0 Command line SDK
 - Xcode 4.0 Cocoa SDK
 - Xcode 4.0 iPhone iOS 4.3 SDK
 - Xcode 4.0 iPhone Simulator iOS 4.3 SDK
 - Xcode 4.0 Plug-in for use with IceTouch
 - Documentation in: /Developer/Documentation/IceTouch-1.1.1
 - Examples in: /Developer/Examples/IceTouch-1.1

To uninstall IceTouch-1.1.1 please use the uninstall script that
accompanies the distribution. 

For uninstall details execute:

> uninstall.sh -h
