This installer installs the following components:

 - Command line Objective-C SDK for Mac OS X in: /opt/IceTouch-1.2.0
 - Xcode 4.3 Cocoa Objective-C SDK in: <Xcode>/Contents/Developer/SDKs/IceTouch-1.2.0/macosx.sdk
 - Xcode 4.3 iPhone iOS 5.0 Objective-C SDK in: <Xcode>/Contents/Developer/SDKs/IceTouch-1.2.0/iphoneos.sdk
 - Xcode 4.3 iPhone Simulator 5.0 Objective-C SDK in: <Xcode>/Contents/Developer/SDKs/IceTouch-1.2.0/iphonesimulator.sdk
 - Xcode 4.3 Cocoa C++ SDK in: <Xcode>/Contents/Developer/SDKs/IceTouchCpp-1.2.0/macosx.sdk
 - Xcode 4.3 iPhone iOS 5.0 C++ SDK in: <Xcode>/Contents/Developer/SDKs/IceTouchCpp-1.2.0/iphoneos.sdk
 - Xcode 4.3 iPhone Simulator 5.0 C++ SDK in: <Xcode>/Contents/Developer/SDKs/IceTouchCpp-1.2.0/iphonesimulator.sdk
 - Xcode 4.3 Plug-in for use with Ice Touch in: /Library/Application Support/Developer/4.3/Xcode/Plug-ins/
 - Documentation in: <Xcode>/Contents/Developer/Documentation/IceTouch-1.2.0

<Xcode> refers to the folder where Xcode 4.3 is installed, /Applications/Xcode.app by default. If you installed Xcode 4.3 in a different location, please adjust the target folder for the "iPhone / Cocoa SDKs" in the next dialog.

To uninstall Ice Touch 1.2.0, please use the uninstall script that accompanies the distribution.
