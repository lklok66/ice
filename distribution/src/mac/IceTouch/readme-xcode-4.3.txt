This installer installs the following components in /Library/Developer/IceTouch-1.2.1:

 - Command line Objective-C SDK for Mac OS X
 - Xcode 4.3 Objective-C SDKs for Mac OS X, iPhone iOS 5.0, iPhone Simulator 5.0
 - Xcode 4.3 C++ SDKs for Mac OS X, iPhone iOS 5.0, iPhone Simulator 5.0
 - Xcode 4.3 Plug-in for use with Ice Touch
 - Documentation

To uninstall Ice Touch 1.2.1, please use the uninstall script that accompanies the distribution.
