This installer installs the following components:

 - Xcode 3.2 Command line SDK in: /opt/IceTouch-1.1
 - Xcode 3.2 Cocoa SDK in: /Developer/SDKs/IceTouch-1.1/macosx.sdk
 - Xcode 3.2 iPhone iOS 4.2 SDK in: /Developer/SDKs/IceTouch-1.1/iphoneos.sdk
 - Xcode 3.2 iPhone Simulator iOS 4.2 SDK in: /Developer/SDKs/IceTouch-1.1/iphonesimulator.sdk
 - Xcode 3.2 Plug-in for use with IceTouch in: /Library/Application Support/Developer/3.2/Xcode/Plug-ins/
 - Documentation in: /Developer/Documentation/IceTouch-1.1
 - Examples in: /Developer/Examples/IceTouch-1.1

To uninstall IceTouch-1.1.1 please use the uninstall script that
accompanies the distribution.

For uninstall details execute:

> uninstall.sh -h
