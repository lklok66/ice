This installer installs the following components:

 - Command line SDK for Mac OS X in: /opt/IceTouch-1.1
 - Xcode 3.2 Cocoa SDK in: <Xcode>/SDKs/IceTouch-1.1/macosx.sdk
 - Xcode 3.2 iPhone OS 4.2 SDK in: <Xcode>/SDKs/IceTouch-1.1/iphoneos.sdk
 - Xcode 3.2 iPhone Simulator 4.2 SDK in: <Xcode>/SDKs/IceTouch-1.1/iphonesimulator.sdk
 - Xcode 3.2 Plug-in for use with Ice Touch in: /Library/Application Support/Developer/3.2/Xcode/Plug-ins/
 - Documentation in: <Xcode>/Documentation/IceTouch-1.1
 - Examples in: <Xcode>/Examples/IceTouch-1.1

<Xcode> refers to the folder where Xcode 3.2 is installed, /Developer by default. If you installed Xcode 3.2 in a different location, please adjust the target folder for the "iPhone / Cocoa SDKs" in the next dialog.

To uninstall Ice Touch 1.1.1, please use the uninstall script that accompanies the distribution.
