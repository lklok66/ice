Ice Touch 1.0.0 offers an Objective-C language mapping, development
kits and run times for Mac OS X and the iPhone OS.

Ice is a modern alternative to object middleware such as CORBA or
COM/DCOM/COM+. It is easy to learn, yet provides a powerful network
infrastructure for demanding technical applications. Ice features
an object-oriented specification language, many different language
mappings, a highly efficient protocol, asynchronous method invocation
and dispatch, dynamic transport plug-ins, TCP/IP and UDP/IP support,
SSL-based security, a firewall solution, and much more.
