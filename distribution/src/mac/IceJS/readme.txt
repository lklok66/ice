Installing Ice for JavaScript 0.1.0

Double-click on IceJS-0.1.0.pkg and follow the instructions to install Ice for JavaScript on your computer.

Removing Ice for JavaScript 0.1.0

To remove Ice for JavaScript 0.1.0 from your computer, execute the uninstall.sh script as root in a Terminal:

  $ cd /Volumes/IceJS-0.1.0
  $ sudo ./uninstall.sh
   
uninstall.sh removes all Ice for JavaScript 0.1.0 installations from your computer.
