Demos in this directory:

- server

  A QuickFIX server to run against the clients.

- simple

  A simple client for IceFIX.

- nonfiltered

  A client that demonstrates the filtered QoS.

- grid

  A demo that shows how to manage multiple IceFIX bridges with
  IceGrid.

- backup

  A demo that shows how to do a hot backup of the IceFIX bridge
  database files.
