    Chat Demo Readme

    This demo is a poll client for the ZeroC's Chat Demo application.

    You can use the Live Client if you don't want to setup your own server.

    First follow the instructions from the Chat Demo Distribution README to start the server.

    When the demo is build with optimizations enabled it creates a minified script browser/Client.min.js that includes:

    • Ice.js (The Ice run-time library)
    • Glacier2.js (The Glacier2 library)
    • Chat.js & ChatSession.js (The generated code for this demo)
    • Client.js (The client application)

    to use the minified version you should edit the demo/ChatDemo/index.html file and comment out the non optimized scripts:

        <!-- Scripts used during development, for optimized builds 
    comment out the following scripts and use browser/Client.min.js below -->
    <script src="../../../lib/Ice.js"></script>
    <script src="../../../lib/Glacier2.js"></script>
    <script src="Chat.js"></script>
    <script src="ChatSession.js"></script>
    <script src="Client.js"></script>

    Then uncomment the script tag for the minified version

        <!-- Uncomment the following line to use use a minimized version of the 
         Client that include Ice and Glacier2 libraries and the Chat Demo client. -->
    <!--<script src="Client.min.js"></script>-->
