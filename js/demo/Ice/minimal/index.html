Minimal Demo Readme

This demo is a minimal Ice "hello world" application. Each time the client is run a "sayHello" invocation is sent to the server.

First follow the instructions from the C++ Ice minimal demo README to start the server with Web Socket protocol enabled.

To run the client click the "Hello World!" button.

Note that this demo hardwires port 10000. If port 10000 is not available on your machine, you need to edit both client and server to use a free port.

Using the minified scripts

When the demo is build with optimizations enabled it creates a minified script browser/Client.min.js that includes:

  • Ice.js (The Ice run-time library)
  • Hello.js (The generated code for this demo)
  • browser/Client.js (The demo source)

to use the minified version you should edit the demo/Ice/minimal/index.html file and comment out the non optimized scripts:

    <!-- Scripts used during development, for optimized builds 
    comment out the following scripts and use browser/Client.min.js below -->
    <!-- Ice.js (Ice run-time library) -->
    <script type="text/javascript" src="../../../lib/Ice.js"></script>
    <!-- Hello.js (Demo generated code) -->
    <script type="text/javascript" src="Hello.js"></script>
    <!-- browser/Client.js (Bidir Demo Application) -->
    <script type="text/javascript" src="browser/Client.js"></script>

Then uncomment the script tag for the minified version

    <!-- Uncomment the following line to use a minified version of the 
         scripts that includes: (the Ice run-time library, the generated code
         and the demo application). -->
    <!--<script type="text/javascript" src="browser/Client.min.js"></script>-->
